RCHome

Аппаратно программный комплекс для управления освещением и наблюдения за основными параметрами в комнатах квартиры

Аппаратная часть:
	SensorBoard сенсорный модуль на основе "Arduino Leonardo" + "nRF24l01+"
		Цели:
		Управление выключателями
		Снятие показанаий датчиков
		Прием команд с ControlBorad по радиоканалу
		Передача показаний датчиков на ControlBorad по радиоканалу


	ControlBoard управляющий модуль на основе "Raspberry Pi" + "nRF24l01+" 
		Цели:
		Периодический сбор показаний датчиков со всех SensorBoard
		Передача управляющих команд на все SensorBoard
		Хранение всех показаний в БД
		Обеспечение API для отображения показаний и управления выключателями
		Обеспечение Web-интерфейса для отображения показаний и управления выключателями
